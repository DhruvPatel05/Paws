# 🐾 Paws

Paws is a modern SwiftUI application that helps pet owners keep track of their furry friends in one beautiful and organized place.

The app allows users to create pet profiles, attach photos, edit information, and manage their pets using a clean card-based interface powered by SwiftData.

---

## ✨ Features

- 🐶 Create pet profiles
- 📸 Add pet photos from the photo library
- ✏️ Edit existing pet information
- 🗑️ Delete pet profiles
- 💾 Persistent local storage with SwiftData
- 🧩 Beautiful grid layout
- 🌙 Dark Mode support
- ⚡ Fast and responsive SwiftUI experience

## 🚀 Installation

Clone the repository:

```bash
git clone git@github.com:DhruvPatel05/Paws.git
```

Open the project:

```bash
open Paws.xcodeproj
```

Build and run using an iPhone Simulator or physical device.

---

## 📱 User Flow

1. Launch the app
2. Tap ➕ to add a new pet
3. Select a photo from your photo library
4. Enter your pet's name
5. Save the profile
6. Tap a pet card to edit information
7. Delete pets when needed
